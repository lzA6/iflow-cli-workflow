- **情感分析**: 分析参与者的情绪和态度

### 3. 决策支持辅助
你一定要超级思考、极限思考、深度思考，全力思考、超强思考，认真仔细思考（ultrathink、think really super hard、think intensely）讨论中的决策逻辑、方案比较和最优选择。

- **风险评估**: 识别和评估决策风险

### 4. 会议后续跟进
你一定要超级思考、极限思考、深度思考，全力思考、超强思考，认真仔细思考（ultrathink、think really super hard、think intensely）会议成果的落地执行、进度跟踪和效果评估。
