## 用法

```bash
python .iflow/commands/arq-analysis-workflow-v9.py [选项]
```

## 🔧 选项

### 基础选项
- `--workspace, -w`: 工作空间路径（默认：当前目录）
- `--detailed`: 详细分析模式
- `--output-format`: 输出格式（json/yaml/markdown，默认：json）
- `--no-save`: 不保存分析结果

### 分析模式
- `--performance`: 仅性能分析
- `--compliance`: 仅合规性检查
- `--optimization`: 仅优化建议
- `--auto-optimize`: 自动执行优化

### V9高级选项
- `--quantum-mode`: 启用量子分析模式（默认启用）
- `--parallel-analysis`: 并行分析（默认启用）
- `--predictive-insights`: 预测性洞察（默认启用）
- `--ai-enhanced`: AI增强分析（默认启用）
- `--multi-agent`: 多智能体协作（默认启用）
- `--real-time`: 实时监控（默认启用）
- `--adaptive`: 自适应优化（默认启用）

### 常见问题
1. **依赖缺失**: 确保V9组件已正确安装
2. **权限问题**: 检查工作空间目录权限
3. **GPU不可用**: V9可在CPU模式下运行，但性能会降低
4. **内存不足**: V9需要更多内存用于AI分析

